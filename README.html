🏉 Rugby School Club - V2 (Modern Edition)

Bem-vindo à segunda iteração do projeto Rugby School Club. Esta versão marca uma evolução significativa no design e na arquitetura do código, focando em performance, responsividade e modernização visual.

🚀 O que há de novo na V2?

Esta atualização não foi apenas estética; ela foi focada em eficiência. A principal mudança foi a migração do CSS tradicional para o Tailwind CSS.
Uso de Gemini IA para criação do ReadMe, pratica que otimiza o trabalho.

💡 Por que Tailwind CSS?

Diferente da V1, onde utilizávamos múltiplos arquivos CSS externos, a V2 utiliza a técnica de Utility-First. Os principais benefícios observados foram:

  • Otimização de Fluxo: Desenvolvimento muito mais rápido diretamente no HTML, sem a necessidade de alternar entre arquivos.
  • Redução de Arquivos: Diminuímos drasticamente o número de folhas de estilo, o que facilita a manutenção e o carregamento do site.
  • Mobile-First Nativo: A facilidade de gerenciar breakpoints diretamente nas classes permitiu um site muito mais fluido em dispositivos móveis.
  • Consistência Visual: Uso de uma escala de design pré-definida (espaçamentos, cores e fontes).

🛠️ Tecnologias Utilizadas

  • HTML5 (Semântico)
  • Tailwind CSS (Estilização via utilitários)
  • Swiper.js (Slider interativo para a seção de Treinadores)
  • Google Fonts (Montserrat para títulos de impacto e Inter para leitura)
  • GitHub Pages (Hospedagem)

📱 Destaques da Versão

  • Menu Mobile: Implementação de menu toggle (hambúrguer) para melhor navegação em telas pequenas.
  • Slider de Coaches: Carousel responsivo que suporta gestos de "swipe" no celular.
  • Hero Imersiva: Design agressivo e moderno com overlays de gradiente, remetendo à energia do Rugby.
  • Bento Grid: Layout de notícias otimizado para diferentes resoluções.

🔧 Como visualizar o projeto

  1. Clone o repositório:
    git clone [https://github.com/josebossle-dev/site-rugby.git](https://github.com/josebossle-dev/site-rugby.git)
